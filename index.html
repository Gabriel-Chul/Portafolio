INICIAR SESIÓN

Este es un proyecto PHP que además utiliza los lenguajes de HTML, CSS, JavaScript, SQL y se ha hecho uso de la libreria de MaterializeCSS. Es un sistema de login de usuarios con registro, base de datos, perfil de usuario, metodo de recuperación de contraseña, y algunas funcionalidades interesantes como una busqueda de los usuarios registrados.

BLOG - MINECRAFT

Este es un proyecto HTML y CSS, utilizando tambien el lenguaje de programación JavaScript para crear cajas de comentarios.

Este es un Website que funciona como un BLOG con la tematica del juego de MINECRAFT. Esta pagina web cuenta con una TIENDA online (Ecommerce), un FORO, y algunos detalles interesantes, como un sistema de donaciones, en la que aparecera el nombre (NICKNAME) de la persona al costado derecho de la pagina web una vez completado el proceso de donación.
